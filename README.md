AgriFund – Plateforme Laravel de Financement Participatif pour Jeunes Entrepreneurs

AgriFund est une plateforme web développée avec Laravel permettant aux jeunes entrepreneurs de présenter leurs projets et de recevoir un financement participatif.
Les utilisateurs peuvent parcourir les projets, acheter des points, puis contribuer aux projets de leur choix.

Le système repose sur une architecture Laravel structurée, des vues Blade personnalisées et une base de données MySQL.
Le projet met en avant des compétences solides en MVC, gestion d’authentification, logique métier, Blade templates et intégration avec une base de données.
